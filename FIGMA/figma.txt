🎯 Корневые файлы
App.tsx
Главная точка входа приложения. Просто рендерит роутер внутри обертки с темной темой и шрифтами.

routes.tsx
Настройка всех маршрутов приложения:

/ → CalendarPage
/profile → ProfilePage
/profile/employees → EmployeesPage
/profile/reports → ReportsPage
/profile/settings → SettingsPage
/profile/settings/admin → AdminPage
/profile/settings/admin/employees → EmployeeManagementPage
/profile/bug-report → BugReportPage
📐 Layout (структура)
MainLayout.tsx
Общая обертка для всех страниц. Содержит:

Нижнюю навигацию (TabBar) с двумя кнопками: Календарь и Профиль
Кнопку "Назад" для подстраниц (например, когда открыт /profile/employees)
Автоматически скрывает TabBar на подстраницах
📄 Pages (страницы приложения)
CalendarPage.tsx ⭐ ГЛАВНАЯ
Календарь смен с цветными ячейками:

Переключение месяцев (свайпом или кнопками)
Кнопка выбора коллег (иконка Users) - открывает ColleaguesModal
Цветовая кодировка смен по департаментам
Клик на день → открывает DayModal с деталями
Индикатор синхронизации с Google Sheets
ProfilePage.tsx
Страница профиля пользователя:

Аватар с инициалами
Имя и должность
Меню: Сотрудники, Отчёты, Настройки, Сообщить об ошибке
EmployeesPage.tsx
Список всех сотрудников:

Поиск по имени/должности
Фильтр избранных (звёздочка)
Добавление в избранное
ReportsPage.tsx
Статистика по часам и доходам:

Текущий месяц: часы и доход
История прошлых месяцев
Карточки с данными
SettingsPage.tsx
Настройки приложения:

Переключатель темной/светлой темы
Ссылка на админ-панель (если права есть)
Кнопка "Выйти из аккаунта"
AdminPage.tsx
Админ-панель (только для админов):

Настройки Google Sheets (URL, API Key, Apps Script URLs)
Индикатор статуса синхронизации
Ссылка на панель управления сотрудниками
EmployeeManagementPage.tsx
Управление заметками о сотрудниках (для админов):

Список всех сотрудников
Возможность добавить/редактировать заметки по каждому
BugReportPage.tsx
Связь с поддержкой:

Красная кнопка "Сообщить об ошибке" → ссылка на личку админа
Синяя кнопка "Предложить улучшение" → ссылка на личку админа
🧩 Components (компоненты)
TabBar.tsx (СТАРЫЙ, не используется)
Нижняя навигация из предыдущей версии. Сейчас навигация реализована в MainLayout.

ThemeToggle.tsx
Переключатель темной/светлой темы (кнопка с иконкой луны/солнца).

ColleaguesModal.tsx
Модальное окно выбора коллег:

Выбор до 3-х сотрудников
Каждому коллеге присваивается свой цвет
После выбора их смены показываются в календаре
DayModal.tsx
Модальное окно деталей дня:

Показывает все смены в выбранный день
Можно редактировать время и добавлять заметки к смене
Смены отсортированы по иерархии департаментов
CalendarView.tsx (СТАРЫЙ, не используется)
Старая версия календаря из предыдущего интерфейса. Можешь использовать как референс или удалить.

ProfileView.tsx (СТАРЫЙ, не используется)
Старая версия профиля. Содержал табы: Сотрудники, Отчёты, Настройки, Связь. Сейчас это отдельные страницы.

figma/ImageWithFallback.tsx
Компонент для изображений с fallback на случай ошибки загрузки.

📁 Папки
components/ui/
Библиотека готовых UI-компонентов (shadcn/ui): кнопки, инпуты, модалки, формы, селекты и т.д. Готовые блоки для быстрой разработки.

imports/
Скриншоты старого интерфейса для референса.

styles/
CSS файлы: глобальные стили, темы, шрифты, Tailwind v4 конфиг.

Для замены интерфейса: редактируй файлы в pages/ - там текущий контент каждой страницы.
